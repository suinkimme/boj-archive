# NEXT JUDGE

## 최백준 님께

먼저 진심으로 사과의 말씀을 드립니다.

저는 오랫동안 acmicpc.net에서 수많은 문제를 풀어왔고, 그 과정에서 알고리즘을 배우고 성장할 수 있었습니다. 그 토대를 만들어주신 최백준 님과 BOJ 팀에 항상 감사함을 느껴왔습니다.

그럼에도 불구하고 BOJ의 서비스 정책을 충분히 확인하지 않은 채 문제 데이터를 수집하고 공개 저장소에 올린 것은 명백히 잘못된 행동이었습니다. "공익적 목적"이라는 이유로 스스로를 합리화했지만, 그것이 저작권자와 출제자 분들의 권리를 침해하는 행위였음을 인정합니다. 변명의 여지가 없습니다.

요청하신 즉시 저장소의 모든 문제 데이터를 삭제하고, git 히스토리에서도 완전히 제거했습니다. 서비스에서의 문제 노출도 즉시 중단했습니다.

앞으로 NEXT JUDGE는 독자적으로 제작한 문제만을 사용하는 방향으로 다시 시작하겠습니다. BOJ와 혼동될 수 있는 어떠한 표현도 사용하지 않겠으며, 올바른 방식으로 서비스를 만들어나가겠습니다. 조금만 기다려 주시면 더 나은 모습으로 돌아오겠습니다.

다시 한번 깊이 사과드립니다.
